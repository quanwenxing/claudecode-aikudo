# TODOアプリケーション

[English](README.md)

和風モダン（Wa-fu Modern）デザインを特徴とする、バニラHTML、CSS、JavaScriptで構築されたスタンドアロンTODOアプリケーションです。

## 機能

- ✨ シンプルで美しい和風デザイン
- 📝 タスクの追加、編集、削除
- ✓ タスクの完了マーク
- 💾 localStorageによる自動データ保存
- 🎨 Zen AntiqueとZen Kaku Gothic Newフォントによる美しいタイポグラフィ

## はじめに

インストールやビルドツールは不要です！ブラウザでファイルを開くだけで使用できます：

```bash
open todo.html
```

または、`todo.html`をダブルクリックしてデフォルトブラウザで開いてください。

## アーキテクチャ

- **シングルファイルアーキテクチャ**: すべてのHTML、CSS、JavaScriptが`todo.html`に含まれています
- **データ永続化**: localStorageを使用（キー: `kokoroecho-todos`）
- **デザインテーマ**: 和風モダン（Wa-fu Modern）の美学

## 技術スタック

- ピュアHTML5
- カスタムプロパティを使用したCSS3
- バニラJavaScript（ES6+）
- 依存関係やフレームワークなし

## ブラウザ互換性

以下をサポートするすべてのモダンブラウザで動作します：
- localStorage
- ES6 JavaScript機能
- CSSカスタムプロパティ
